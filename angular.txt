- a configuração das rotas se da atraves dos metodos when e otherwise do serviço $routeProvider

.config(function($routeProvider) {
	$routeProvider
	.when()
	.otherwise()	
})

- a função when é a responsavel em configurar as rotas e ela possui algumas funçoes uteis como templateUrl, controlle, controllerAs e etc...(05:30)

-ex pratico de como ficara uma rota

.config(function($routeProvider) {
	$routeProvider
	.when('/users', {  caminho apos o endereço
	templateUrl: 'users.html',  caminho pasta
	controller: 'UserController',
	controllerAs: 'User'
	})
	.otherwise({
	redirectTo: '/index'   se nao tiver nada vai redirecionar pra ca
	})
})

ex de como ficara a rota:

.config(function($routeProvider) {
	$routeProvider
	.when('/users', {  caminho apos o endereço
	templateUrl: 'users.html',  caminho pasta
	controller: 'UserController',
	controllerAs: 'User'
	})
	.otherwise({
	redirectTo: '/index'   se nao tiver nada vai redirecionar pra ca
	})
	.when('/users/:id', {  caminho apos o endereço passando o id via parametro, observe, para passar parametro uso :
	templateUrl: 'users.html',  caminho pasta
	controller: 'UserController',
	controllerAs: 'User'
	})
})
